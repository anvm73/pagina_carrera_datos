Primer prototipo de web en Streamlit: testo.py

Este fue el primer front del CE ICCD en Streamlit: navegación lateral, avisos, enlace al Drive académico, cards del equipo y un corazón 3D con Plotly. Hoy usamos esta landing en Astro para el público, conservando el código Python como referencia y descarga.

Descargar .zip (testo.py + requirements.txt) Descargar solo testo.py

¿Qué problema resuelve?

  • Centraliza comunicación: bienvenida, avisos, Drive y contacto.
  • Muestra resultados técnicos: visualización 3D con Plotly y NumPy.
  • Renderiza equipo con imágenes remotas (requests + Base64) dentro de Streamlit.

Tecnologías

  • Streamlit para UI y navegación.
  • Plotly (Surface) para el corazón 3D.
  • NumPy y Matplotlib para cálculo y visualización.
  • Requests + Base64 para obtener logos e imágenes desde GitHub.

Instalación local

python -m venv .venv
source .venv/Scripts/activate  # en Windows: .venv\\Scripts\\activate
pip install -r requirements.txt
streamlit run testo.py

Casos de uso

  • Mostrar avisos y eventos del CE ICCD.
  • Dar acceso al Drive académico.
  • Divulgar el equipo y enlaces de contacto.
  • Presentar una visualización interactiva (corazón 3D) como demostración técnica.

Archivos incluidos

  • src/assets/python/testo.py: script completo.
  • src/assets/python/requirements.txt: dependencias.
  • public/downloads/python-code.zip: descarga directa.

Para la landing usamos Astro + Tailwind; el código Python se presenta como referencia y no se ejecuta en el navegador.
