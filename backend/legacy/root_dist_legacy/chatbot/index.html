Chatbot

Chatbot del Centro de Estudiantes

Conversa con el asistente del Centro de Estudiantes para resolver dudas frecuentes sobre la carrera, actividades, malla y datos de contacto. El chat se actualiza en tiempo real y verás las respuestas escribirse letra por letra.

CE
¡Hola! 👋 Soy el chatbot del Centro de Estudiantes de Ciencia de Datos UTEM. Escríbeme tus dudas sobre la carrera, el CE, actividades, contacto u otra consulta relacionada.
El chatbot utiliza un modelo local (Ollama) y documentos ICCD. Para usarlo, asegúrate de tener el servidor de Python del chatbot encendido.

Sobre este chatbot

  • • Entrenado con documentos de la carrera y material del CE.
  • • Ideal para dudas rápidas de malla, contacto y actividades.
  • • Corre localmente usando un modelo de lenguaje vía Ollama.

Recomendaciones de uso

  • • Formula preguntas claras y concretas.
  • • Evita compartir datos personales sensibles.
  • • Usa el correo oficial para trámites formales.
